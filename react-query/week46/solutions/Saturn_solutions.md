2. refetchOnMount는 어떤 상황일 때 설정하는 옵션인가요?

- 쿼리의 새 인스턴스가 마운트될 때

  컴포넌트가 처음 렌더링될 때,
  컴포넌트가 언마운트 후 다시 마운트될 때,
  쿼리 키가 변경될 때 등
