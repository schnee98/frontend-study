_답_
```

```

2. 디음 코드의 this는 어느 this를 가리키는지 설명하시오.

```js
let value = 1;
const obj = {
    value: 100,
    foo() {
        setTimeout(() => console.log(this.value), 100); // 여기 this
    },
};
```
_답_
```
상위 스코프의 this
```
