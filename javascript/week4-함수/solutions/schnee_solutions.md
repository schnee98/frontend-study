3. 다음 함수들의 차이점을 말해주세요.
```
function add(x, y) {
  return x + y;
}

var sub = (x, y) => x - y;
```

답:
- `add()` 함수는 함수 호이스팅에 의해 생성되고 선언되기 때문에 선언문 이전에 호출이 가능하다.
- `sub()` 함수는 변수 호이스팅에 의해 `sub` 변수가 생성되어 `undefined`로 초기화 되기 때문에, 선언문 이전에 호출이 불가능하다.
