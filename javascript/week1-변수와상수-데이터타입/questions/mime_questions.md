## 문제

1. 값을 저장하기 위한 메모리 공간을 확보하고 변수 이름과 확보된 메모리 공간의 주소를 연결해서 값을 저장할 수 있게 **준비**하는 것을 무엇이라 하는가?

2. 이하 참조 에러가 나는 것은?

```
fn1(); // (1) 함수 선언
fn2(); // (2) 함수 표현식

function fn1() {
	console.log("fn1");
}

const fn2 = () => {
	console.log("fn2");
};
```

3. 값이 없다는 것을 의도적으로 명시할 땐 변수에 어떤 것을 할당하는가?

4. 위의 답을 변수에 할당했을때 해당 변수는 어떻게 되는가?

5. 변수 이름과 변수값은 어디에 어떤 형식으로 저장되는가?
