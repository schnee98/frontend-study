_답_
```
디바운싱: 연이어 호출되는 함수들 중 마지막 함수(또는 제일 처음)만 호출하도록 하는 것
        ajax 검색 (검색어 자동완성)
        짧은 간격으로 호출되면 타이머 재설정. 일정 시간 동안 입력이 없으면 완료로 판단. 콜백함수 호출
스로틀 : 마지막 함수가 호출된 후 일정 시간이 지나기 전에 다시 호출되지 않도록 하는 것
        스크롤 이벤트
        짧은 시간 간격으로 발생하는 과도한 이벤트 호출을 막기 위해 호출 주기를 만든다.
 ```

2. Ajax란?

_답_
```
자바스크립트를 사용해여 브라우저가 서버에게 비동기 방식으로 데이터를 요청하고, 서버가 응답한 데이터를 수신하여 
웹페이지를 동적으로 갱신하는 프로그래밍 방식을 말한다. 서버로부터 웹페이지의 변경에 필요한 데이터만 비동기 방식으로 전송받아 
웹페이지를 변경할 필요가 없는 부분은 리렌더링하지 않고 부분적으로만 렌더링이 가능하다.
```

3. URI란?
_답_
```
Uniform Resource Identifier 식별자 url의 상위 개념
url은 위치 urn은 이름(id같이 판별 가능한 이름)

https://ㅇㅇㅇ.tistory.com/category

hstory0208.tistory.com 에서 category 라는 경로를 나타냅니다.
category는 리소스의 실제 위치이므로 이 주소는 URL 입니다.
 

https://ㅇㅇㅇ.tistory.com/category/12

ㅇㅇㅇ.tistory.com 에서 category 라는 자원의 경로를 나타내는 부분까진 URL 이지만
/12 는 식별자 이므로 https://ㅇㅇㅇ.tistory.com/category URL을 포함한 URI라고 할 수 있습니다.
```
