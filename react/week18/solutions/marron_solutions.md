3. zustand 사용한 상태관리에 대해 간략하게 서술해주세요.
   _답_

```
하나의 스토어를 중앙 집중형으로 활용해 스토어 내부에서 상태를 관리한다.
Zustand는 발행/구독 모델을 기반으로 이루어져 있다. 스토어의 상태 변경이 일어날 때 실행할 리스너 함수를 모아두었다가(구독) 상태가 변경되었을 때 등록된 리스너들에게 상태가 변경되었다고 알려준다(발행).
```
